# 长条款细切分与重复块降权首包回提

## 文档目的

本文件用于记录“长条款细切分与重复块降权”首包的实现范围、真实样例联调结果和当前正式判断。

## 本轮范围

本轮严格限定在以下范围内：

1. 面向长条款建立更稳的细切分策略
2. 对明显重复块、相邻重叠块、正文与偏离表平行区域重复做抑制
3. 让审查输入更接近可独立判断的最小条款片段
4. 在不破坏现有结果归并口径的前提下重新压降原始命中数量
5. 使用 `SZCG2025000300-A` 重新联调验证压降效果

本轮不处理：

1. 页面扩展
2. 流程扩展
3. 提示词与规则包继续扩张
4. 结果层继续堆复杂归并补丁

## 本轮实现

### 1. 长条款细切分

已在 `app/parser_worker.py` 增加进入审查前的最小条款细切分逻辑。

当前细切分策略如下：

1. 对超长条款按子项编号切分
2. 对单行但包含大量 `|` 分隔内容的长条款做细切分
3. 对切分后的条款片段追加 `片段2`、`片段3` 等定位后缀
4. 缩短过长的 `location_label` 锚点，避免定位字段承载整段正文

### 2. 重复块抑制

已在同一解析阶段增加重复块抑制。

当前抑制策略如下：

1. 对进入审查前的片段做空白归一化
2. 归一化后完全相同的片段只保留一次
3. 因正文 / 偏离表 / 表格格式差异仅在空白上不同的重复片段，会被直接抑制

本轮没有做复杂相似度去重，仍然保持 V1 极简边界。

### 3. 不回头改结果层口径

本轮保持第二优先级已落地的风险组口径不变：

1. 总体结论仍按风险组生成
2. 结果页仍按风险组展示
3. Markdown 报告仍按风险组明细输出

本轮只从输入层压降重复，不继续在结果层叠加补丁。

## 自动化验证

本轮新增并通过以下验证：
