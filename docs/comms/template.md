# 通信消息模板

请按以下模板追加消息。

```md
## 2026-03-28 14:30 T -> 总负责人

### 主题
上传接口是否需要统一文件预处理

### 背景
当前准备进入上传链路实现，PDF 与 Word 后续处理路径存在差异。

### 问题
V1 是否要求上传后先统一进入同一种内部格式，再进入后续审查链路？

### 备选方案
1. 分别处理 PDF / Word
2. 统一转成内部中间格式后再处理

### 当前建议
建议选 2，后续状态和解析链路更统一。

### 期望回复
请总负责人拍板。
```

## 模板要求

- 标题行写清时间、发送方和接收方
- 主题尽量一句话说清
- 背景只写必要信息
- 问题要可回答，不要过于发散
- 如果有建议，尽量提供 1 到 2 个备选方案
